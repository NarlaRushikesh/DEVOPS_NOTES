# 📡 Nagios Practicals List

---

## ✅ Q1 – Nagios Installation and Setup

### 📝 Task:
- Install Nagios Core on the server  
- Configure basic setup  
- Start and enable Nagios service  
- Access Nagios web interface  

---

## ✅ Q2 – Host Monitoring

### 📝 Task:
- Add a new host in Nagios configuration  
- Define:
  - Host name  
  - IP address  
- Verify host status in Nagios dashboard  

---

## ✅ Q3 – Service Monitoring

### 📝 Task:
- Add a service to an existing host  
- Monitor services like:
  - HTTP  
  - SSH  
  - Ping  
- Verify service status in dashboard  

---

## ✅ Q4 – NRPE Configuration

### 📝 Task:
- Install NRPE on client (agent) machine  
- Configure NRPE to communicate with Nagios server  
- Allow monitoring of remote system metrics  

---

## ✅ Q5 – Custom Plugin Setup

### 📝 Task:
- Create or use a custom Nagios plugin  
- Add it to Nagios configuration  
- Execute and verify plugin output  

---

## ✅ Q6 – Web Server Monitoring

### 📝 Task:
- Configure Nagios to monitor web server (Apache/Nginx)  
- Check HTTP service status  
- Verify alerts and status  

---

## 🔥 Summary

- Q1 → Installation & setup  
- Q2 → Host monitoring  
- Q3 → Service monitoring  
- Q4 → NRPE (remote monitoring)  
- Q5 → Custom plugin  
- Q6 → Web server monitoring
